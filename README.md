How to Apply for Police Clearance in the Philippines (Complete Guide)

This repository provides a complete, step-by-step guide on how to apply for a Police Clearance in the Philippines through the **National Police Clearance System (NPCS)**.  
It includes requirements, online application, payment process, and how to resolve “HIT” status issues.

- Police Clearance Requirements:
- 2 valid IDs  
- NPCS Reference Number  
- Online appointment  
- Payment confirmation  
- Biometrics (fingerprint & photo)

- NPCS Online Application Steps
1. Visit the official NPCS portal  
2. Create an account  
3. Validate your email  
4. Update your profile  
5. Choose a police station  
6. Set an appointment  
7. Pay via GCash / LandBank / BancNet  
8. Visit the station for biometrics  
9. Claim your clearance  

- How to Fix “HIT” Status:
A HIT status means your name matches someone with a case record.  
To resolve this:
- Wait for verification (1–2 weeks usually)
- Bring additional valid IDs
- Cooperate with the police verification officer

- Why You Need Police Clearance:
- Job applications  
- Government documents  
- Travel requirements  
- Local employment  
- Personal records validation
